# Snake Game 

A modern Snake Game built with Python and Pygame featuring multiple gameplay enhancements, achievements, power-ups, obstacles, statistics tracking, and a polished user interface.

## Features

### Core Gameplay

* Classic Snake mechanics
* Score tracking
* High score system
* Pause and resume functionality
* Game over and restart system

### Difficulty Levels

* Easy
* Medium
* Hard

### Advanced Features

* Obstacles
* Bonus food
* Power-ups
* Achievement system
* Statistics tracking
* Save and load functionality
* Multiple menus
* Sound effects and music support

### Statistics

* Highest Score
* Total Games Played
* Average Score
* Total Food Eaten

### Achievements

* First Bite
* Hungry Snake
* Score Milestones
* Survival Achievements

## Technologies Used

* Python 3.x
* Pygame

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/snake-game-pygame.git
cd snake-game-pygame
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the game:

```bash
python main.py
```

## Controls

| Key     | Action     |
| ------- | ---------- |
| ↑ ↓ ← → | Move Snake |
| P       | Pause      |
| R       | Restart    |
| ESC     | Exit       |
